# SAM PRO Mobile - React Native App

حولت تطبيق الويب إلى تطبيق موبايل كامل باستخدام React Native و Expo.

## التحول من الويب إلى الموبايل

### ما تم تحويله:

1. **واجهة المستخدم** - من HTML/CSS/JS إلى React Native components
2. **التنقل** - من روابط HTML إلى React Navigation (Tabs + Stack)
3. **قاعدة البيانات** - من LocalStorage إلى Supabase (PostgreSQL)
4. **المصادقة** - من نظام بسيط إلى Supabase Auth كامل مع RLS

### التقنيات المستخدمة:

- **React Native** - إطار العمل الأساسي
- **Expo** - لتبسيط التطوير والنشر
- **TypeScript** - للكتابة الآمنة
- **Supabase** - قاعدة البيانات والمصادقة
- **React Navigation** - للتنقل بين الشاشات

### الميزات المنجزة:

✅ **المصادقة** - تسجيل الدخول والتسجيل
✅ **لوحة التحكم** - إحصائيات شاملة
✅ **العملاء** - إضافة، عرض، بحث، حذف
✅ **الموردين** - إدارة كاملة
✅ **الأصناف** - تتبع المخزون مع تنبيهات
✅ **الفواتير** - عرض قوائم الفواتير
✅ **دعم RTL** - واجهة عربية كاملة
✅ **Row Level Security** - أمان محكم

### للتشغيل:

```bash
npm install
npm start
```

ثم امسح QR code بتطبيق Expo Go على هاتفك.

### للبناء:

```bash
# Android:
npm run android

# iOS:
npm run ios
```

### هيكل المشروع:

```
├── app/                 # شاشات التطبيق
│   ├── (tabs)/         # شاشات التبويبات الرئيسية
│   ├── customers/      # شاشات العملاء
│   ├── suppliers/      # شاشات الموردين
│   ├── products/       # شاشات الأصناف
│   └── invoices/       # شاشات الفواتير
├── lib/                # مكتبات التطبيق
│   ├── supabase.ts    # إعداد Supabase
│   ├── auth.ts        # وظائف المصادقة
│   └── database.ts    # وظائف قاعدة البيانات
└── hooks/              # React hooks مخصصة
    └── useAuth.tsx    # hook للمصادقة
```

### ملاحظات مهمة:

1. التطبيق يدعم اللغة العربية بشكل كامل (RTL)
2. جميع البيانات مخزنة في Supabase بشكل آمن
3. كل مستخدم يرى بياناته فقط (RLS)
4. التطبيق متوافق مع Android و iOS
